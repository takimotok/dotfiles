## Prerequisites

- macOS


## Installation

Run `install.sh` by curl:

```sh
$ curl -L raw.githubusercontent.com/takimotok/dotfiles/main/install.sh | bash
```

Alternatively run the commands below:

```sh
$ git clone https://github.com/takimotok/dotfiles.git
$ chmod u+x install.sh
$ ./install.sh
```
